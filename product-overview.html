! 痛點：想用機器人，但導入太慢太複雜

客戶知道機器人可以提升效率，但從評估到落地動輒 6–12 個月。整合工程量大、風險高、缺乏技術團隊，結果一直拖住唔做。

自己整合（6–12 個月）

請團隊、讀 API 文檔、寫翻譯層、測試聯調 — 仲要承擔失敗風險。大部分客戶直接放棄。

用我哋平台（4–8 週）

接入現有 WMS → Ontology 自動翻譯 → 即刻驅動機器人。唔使換系統、唔使請人。

◈ 定位：第五類 — Integration + Automation Layer

市場四大類玩家各有強項，但都要求客戶在「速度」和「靈活性」之間做取捨。 查看行業象限圖 →

• Full Stack

自研 WMS + 自研機器人。海康、快倉、海柔、曠視。
問題：要換晒你嘅 WMS。

• Robot + Integration

做機器人，對接第三方 WMS。極智嘉、凱樂士。
問題：只接自己嘅機器人。

• WMS Only

純軟件 WMS。聚水潭、旺店通、富勒。
問題：冇機器人能力。

• Robot Only

純機器人，靠對接第三方。牧星、靈動。
問題：冇倉庫業務邏輯。

我哋嘅位置

唔做機器人硬件，唔取代 WMS。
接入客戶現有 WMS → 建統一 Ontology → 驅動任何品牌嘅機器人 → 全流程自動化。
客戶唔使換系統，加一層就可以用機器人做搬運、揀貨、分揀。

! 競爭格局：高速增長嘅新品類

Gartner 定義咗 Multiagent Orchestration Platforms 品類 — robot-agnostic、WMS-agnostic 嘅倉儲中間層。市場年增長率 138%，預測 2032 年達 $5.23B。

已有 7+ 間公司入場（SVT Robotics、GreyOrange、JASCI、Roboteon 等），主要喺北美/歐洲。中國市場仍係空白。

我哋嘅差異化

現有玩家做 connector / middleware（接線層），我哋做 ontology 語義層（理解層）+ 專注中國市場。
