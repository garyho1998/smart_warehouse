## 大倉成本更高嘅原因

唔止係多機器人，成本係非線性增長：

| 成本項 | 小倉 | 大倉點解貴 |
|---|---|---|
| 機器人數量 | 5–15 台 | 50–200+ 台 |
| WMS 軟件 | 基礎版 | 多層調度、波次策略、多區域協同 |
| 網絡基建 | 幾個 AP | 全倉 WiFi 6、5G 專網、定位信標 |
| 充電站/維護 | 1–2 個 | 10+ 個充電站、備件庫、現場工程師 |
| 地圖/路徑規劃 | 簡單 | 多樓層、多區域、交通管制複雜度指數級增長 |
| 整合工作 | 對接 1 個系統 | 對接 WMS + ERP + MES + 輸送線 + 分揀機 |
| 部署週期 | 4–8 週 | 3–12 個月 |
