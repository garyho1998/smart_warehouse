### 虧損原因分析

帳面淨虧損 ¥8.32 億 vs 調整後虧損 ¥0.92 億 — 大部分係優先股公允價值變動（非現金）。

錢去咗邊（2024 年）：
```
營收          ¥24.09 億   ████████████████████████  100%

營業成本      ¥15.70 億   ███████████████▋          65%    ← 硬件物料+製造+部署
                          剩 ¥8.40 億毛利

銷售及營銷    ¥4.60 億    ████▋                     19%    ← 40 國銷售團隊 + 展會
研發          ¥2.82 億    ██▊                       12%    ← 機器人 + AI + 軟件
管理          ¥1.76 億    █▊                        7%     ← 行政 + 40 國運營
                          ─────────────
三項費用合計  ¥9.18 億                              38%

毛利 - 費用 = 8.40 - 9.18 = -¥0.78 億  ← 經營層面幾乎打平
```

**核心結論：生意本身 unit economics 係 profitable 嘅（毛利 35–39%），虧損係全球化擴張嘅代價。**

| 虧損原因 | 具體表現 |
|---|---|
| 全球銷售網絡 | 40+ 國家養銷售團隊、本地倉、售後，銷售費用佔收入 19% |
| 研發投入 | AMR/ACR/叉車/人形機器人多條產線，研發費用佔收入 12% |
| 優先股公允價值 | 上市前融資嘅優先股按市值重估，帳面虧 ¥8.32 億 vs 調整後虧 ¥0.92 億 |

### 關鍵結論

1. **軟件毛利高但燒錢** — SaaS WMS 毛利率 55–68%，但銷售+研發吃掉 50%+，淨利幾乎為零
2. **機器人毛利低但量大** — AMR 毛利率 35–40%，但需要巨額研發同全球化投入
3. **唯一盈利嘅係海康** — 靠母公司供應鏈壓低硬件成本 + 現成銷售渠道
4. **RaaS 模式被放棄** — 極智嘉主動收縮 RaaS（14% → <1%），租賃模式暫時唔 work
5. **生意本身 unit economics 係 work 嘅** — 虧損係擴張策略嘅代價，唔係商業模式缺陷
